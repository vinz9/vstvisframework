VST Visualisation FrameWork
----------------------
- by Vincent Houze.

(http://www.foliativ.net)

Description
-----------
FrameWork for making VST to display visuals synchronized with audio.
Quite a few bits of code taken from different places, so thanks.
Originals authors/licenses were left intact.

Usage
-----
Look at the examples scenes.
